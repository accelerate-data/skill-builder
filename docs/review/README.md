# Code Review Feedback

This directory contains adversarial review feedback documents generated by the `pr-code-reviewer` agent.

Each document is named `<pr-number>-<short-pr-title>-<yyyy-mm-dd>.md` and contains a structured review of the PR against its plan, design document, Linear issue, and functional spec.

## Generating a Review

Run the `pr-code-reviewer` agent in OpenCode and provide a PR URL, number, or branch name.
